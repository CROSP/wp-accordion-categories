Accordion Categories is widget that allows you to display categories list

== Description ==

Accordion Categories allows you to show categories menu in a sidebar. Each widget can be configured according to your needs.
This widget helps visitors of your site to navigate through categories easily. 

Major features in Accordion Categories include:

* Display categories hierarchically.
* Bootstrap menu style.
* Show/hide empty categories (without any post).
* Show/hide category post count (number of post associated with category) including all children
* Category icon
* Extra field on category configuration page for specifying font-awesome icon

= Contribution =

== Installation ==

1. Upload the entire `accordion-categories` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use `Accordion Categories Menu` widget in any registered sidebar `Appearance->Widgets`
